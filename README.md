# GIFs Gallery

A simple academic project to practice React, file uploads, and state management. This app allows users to upload GIFs from their computer and view them in a gallery.

## Features
- Upload GIF files from your computer
- View uploaded GIFs in a gallery
- Responsive UI
- Practice with React state and file handling

## Tech Stack
- React
- TypeScript
- Vite
- Tailwind CSS

## Getting Started

### Prerequisites
- Node.js (v18+)
- npm or yarn

### Installation
```bash
git clone https://github.com/IgnacioSotelo5/gifs-gallery.git
cd gifs-gallery
npm install
# or
yarn install
```

### Running the App
```bash
npm run dev
# or
yarn dev
```

The app will be available at `http://localhost:5173`.
